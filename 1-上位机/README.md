# WS2812B LED矩阵控制上位机

## 功能
- 图形化控制16x16 WS2812B LED矩阵
- 串口通信控制STM32
- 实时绘制LED颜色
- 预设图案
- 亮度限制保护

## 安装依赖
```bash
pip install -r requirements.txt
```

## 运行
```bash
python ws2812b_control.py
```

或双击 `run.bat`

## 使用说明

### 1. 连接串口
- 选择正确的COM端口
- 波特率设置为115200
- 点击"连接"按钮

### 2. 绘制LED
- 在左侧16x16网格上点击或拖动绘制
- 右侧选择画笔颜色
- 支持快捷颜色选择

### 3. 发送数据
- 点击"发送到LED"按钮
- 数据将通过串口发送到STM32

### 4. 预设图案
- 渐变：彩虹渐变效果
- 棋盘：黑白相间
- 边框：绿色边框
- 十字：橙色十字
- 笑脸：简单笑脸图案

## 串口协议
- 同步字节：0xAA
- 数据：768字节（256 LED × 3字节RGB）
- 顺序：行优先，data[row*16+col]

## 注意事项
- 首次使用请先连接串口
- 亮度限制可防止电源过载
- 建议使用5V 10A以上电源
